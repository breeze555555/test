# 🌌 手势特效切换器 2.0 

这是一个基于 **MediaPipe Hands** 的 Web 应用，通过手势捕捉，驱动绚丽的粒子星空、AR 相册及空间交互。

## 🛠️ 技术栈

- **MediaPipe Hands**: 手部实时追踪与 21 个关键点定位。
- **Canvas 2D**: 极致优化的星芒粒子系统与 AR 图像渲染。
- **WebGL**: 提供底层渲染支持。

## 🚀 如何使用

1. **启动服务**：必须在本地 HTTP 环境下运行（例如 VS Code Live Server，或 `npx serve .`），直接用 `file://` 打开无法获取摄像头权限。
2. **浏览器建议**：推荐使用 **Chrome** 或 **Edge**。
3. **切换模式**：在画面左上角的模式面板，用**食指指向对应模式行并停留 0.8 秒**即可无缝切换。
4. **环境准备**：确保光线充足，手掌面向摄像头，保持身体在画面范围内。

## ✨ 模式交互指南

### Mode 1：Breeze 专属星空 (唯美粒子交互)

这是一个沉浸式的 3D 星空体验，不再有丑陋的轨迹，取而代之的是纯粹的星芒与粒子流。

| **手势**         | **交互效果**                                                 |
| ---------------- | ------------------------------------------------------------ |
| **五指张开**     | **呼唤星风**：漫游星辰汇聚，在掌心上方拼出流光溢彩的 **"Breeze"** 字样。 |
| **单指 (食指)**  | **指尖星环**：星辰化作引力场，随指尖轨迹形成唯美的环形漩涡。 |
| **紧握拳头**     | **超新星爆发**：星辰瞬间向外炸开，形成极具冲击力的视觉效果。 |
| **双手同时出镜** | **空间缩放**：像拉手风琴一样开合双手，可自由拉近观赏星尘细节或俯瞰浩瀚宇宙。 |

### Mode 2：AR 照片画廊 (相册轮播)

将原本的“取景框”升级为 AR 展示模式。

| **手势**         | **交互效果**                                                 |
| ---------------- | ------------------------------------------------------------ |
| **双手捏合四角** | **召唤画廊**：四指框选区域，系统将照片投射至框选区域内，支持自动等比裁剪。 |
| **快速捏合**     | **照片切换**：在框选状态下，做一次“快速捏合”动作，自动切换到下一张照片。 |

### Mode 3：3D 粒子牵引

经典的力学交互模式。

- **五指张开**：生成粒子。
- **单食指伸出**：食指尖形成引力场，吸引粒子跟随移动。
- **双手食指交叉**：当粒子汇聚过半时，触发爆破。

### Mode 4：隔空手写

在这个模式下，你可以在空中实时书写，并能直接操作笔画。

- **拇指食指捏合**：书写/绘画。
- **食指悬停 1.5 秒**：进入移动模式，可拖动已有笔画。
- **双食指缩放**：缩放所有笔画。
- **三指手势**：清空画布，笔画消散为粒子。

## 🔧 常见问题与提示

- **关于镜像**：界面已经做了镜像翻转处理（`scale(-1, 1)`），符合直觉（右手对应屏幕右侧）。
- **关于文字镜像**：代码内部已自动进行反向抵消处理，你会看到正确的 "Breeze" 字样。
- **添加照片**：请在 `ViewfinderMode` 的 `photoUrls` 数组中修改路径（如 `['./img/1.png', './img/2.png']`），确保图片文件已放在项目中。
- **性能优化**：目前粒子系统已采用**离屏纹理缓存**与**批处理渲染**，在 5000 颗星芒同屏下依然能保持 60FPS。若感觉卡顿，请关闭其他高负载网页。

> **给开发者的小贴士**：若需要调整星空的颜色，可在 `ParticleMode` 的 `init()` 方法中修改 `colors` 数组；若想更换“Breeze”文字，在 `_parseTextToTargets` 中修改文字内容即可。
